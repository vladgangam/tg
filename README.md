# CalcUA Bot

CalcUA Bot - простий Telegram-бот українською мовою для щоденних побутових розрахунків.

## Можливості

- `/start` - меню з кнопками.
- Розрахунки можна запускати кнопками: натисніть кнопку, потім введіть потрібні числа через пробіл.
- `/fuel км розхід_л_на_100 ціна_за_літр` - розрахунок пального.
- `/electric старий_показник новий_показник тариф` - розрахунок електроенергії.
- `/water старий_показник новий_показник тариф` - розрахунок води.
- `/unit ціна кількість` - ціна за 1 кг, 1 л або 1 шт.
- `/credit сума відсоток_річних місяців` - орієнтовний ануїтетний платіж за кредитом.
- `/split сума кількість_людей` - розподіл суми між людьми.
- Дані користувачів зберігаються в SQLite у файлі `calcua_bot.db`.

## Встановлення

1. Створіть віртуальне середовище:

```bash
python -m venv .venv
```

2. Активуйте його.

Windows PowerShell:

```powershell
.\.venv\Scripts\Activate.ps1
```

Linux / macOS:

```bash
source .venv/bin/activate
```

3. Встановіть залежності:

```bash
pip install -r requirements.txt
```

4. Створіть файл `.env` на основі `.env.example`:

```env
BOT_TOKEN=ваш_токен_від_BotFather
```

5. Запустіть бота:

```bash
python bot.py
```

## Приклади команд

Після натискання кнопки можна вводити тільки числа, без команди:

```text
371 5.5 74
1250 1320 4.32
180 3.5
```

Команди теж залишаються доступними:

```text
/fuel 371 5.5 74
/electric 1250 1320 4.32
/water 100 112 30.38
/unit 180 3.5
/credit 50000 24 12
/split 1200 4
```

## Примітка

Розрахунок кредиту є орієнтовним і використовує стандартну формулу ануїтетного платежу. Реальні умови банку можуть включати комісії, страхування та інші платежі.
